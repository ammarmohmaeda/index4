تحليل التغيرات المورفولوجية

Morphological Changes Analysis

يقدم التطبيق نظامًا تفاعليًا لتحليل التغيرات المورفولوجية للبحيرات الشمالية في دلتا مصر (البرلس، المنزلة، إدكو، مريوط) بدقة ومرونة. يتضمن الوظائف التالية: خريطة تفاعلية عرض دقيق لمواقع البحيرات باستخدام إحداثيات جغرافية. دعم طبقات خرائط متعددة (OSM، Esri، Google Satellite) لعرض مرن. تحليل شبكات الطرق إمكانية رسم شبكات طرق مخصصة. إحصائيات عامة تتضمن إجمالي المساحة وأكبر وأصغر بحيرة سنويًا. رسوم بيانية متنوعة (خطية، دائرية، أعمدة) توضح تغيرات المساحات لكل بحيرة وللمجموعة من 2013 إلى 2025.

The application provides an interactive system for accurately and flexibly analyzing morphological changes in the northern lakes of Egypt's Delta (Burullus, Manzala, Edku, Mariout). It includes: Interactive Map: Precise display of lake locations using geographical coordinates with multiple map layers (OSM, Esri, Google Satellite). Road Network Analysis: Custom road network drawing capability. General Statistics: Total area and annual largest/smallest lake statistics. Various Charts: Line, pie, and bar charts showing area changes for each lake and the group from 2013 to 2025.

الغطاءات الأرضية Land Cover

الغطاءات الأرضية لسطح الأرض

Land Cover Classification

يقدم التطبيق نظامًا تفاعليًا متطورًا لتحليل تغيرات الغطاء الأرضي باستخدام بيانات Sentinel-2 بدقة عالية ومرونة فائقة. الوظائف الرئيسية: خريطة تفاعلية: عرض دقيق للغطاء الأرضي باستخدام إحداثيات جغرافية واضحة. تحليل زمني ديناميكي: متابعة التغيرات السنوية في الغطاء الأرضي من 2017 إلى 2025. أداة ValuePicker التفاعلية لتحديد السنة وعرض التغيرات تلقائيًا. تصدير الخرائط: تحميل الخريطة كصورة بصيغة PNG بنقرة واحدة عبر زر مخصص.

The application provides an advanced interactive system for analyzing land cover changes using Sentinel-2 data with high precision and flexibility. Key Features: Interactive Map: Precise land cover display with clear geographic coordinates. Dynamic Temporal Analysis: Tracking annual land cover changes from 2017 to 2025. ValuePicker tool for automatic year selection and change display. Map Export: One-click PNG image download via dedicated button.

استشعار Remote Sensing

تحليل صور الأقمار الصناعية

Satellite Image Analysis

يقدم التطبيق نظامًا تفاعليًا متقدمًا لتحليل بيانات صور Landsat الشهرية بدقة عالية ومرونة فائقة. الوظائف الرئيسية: خريطة تفاعلية: عرض دقيق لبيانات Landsat باستخدام إحداثيات جغرافية واضحة. تحليل زمني شهري: متابعة التغيرات في الغطاء الأرضي شهريًا من عام 2000 إلى 2025. اختيار النطاقات الطيفية: دعم تحليل النطاقات الطيفية المتنوعة (مثل الألوان الطبيعية، الأشعة تحت الحمراء، والنطاقات الفردية من 1 إلى 11). أدوات الرسم والقياس: رسم النقاط، الخطوط، والمضلعات على الخريطة. حساب المساحات (بالهكتار) والأطوال (بالمتر) تلقائيًا وعرضها في واجهة المستخدم.

The application provides an advanced interactive system for analyzing monthly Landsat imagery with high precision and flexibility. Key Features: Interactive Map: Precise Landsat data display with clear geographic coordinates. Monthly Analysis: Tracking monthly land cover changes from 2000 to 2025. Spectral Bands: Support for various spectral bands (natural color, infrared, individual bands 1-11). Drawing Tools: Points, lines, and polygons drawing with automatic area (hectares) and length (meters) calculations.

نظام الرصد Monitoring System

نظام الرصد المتكامل للبحيرات الشمالية (2013-2035)

Integrated Monitoring System for Northern Lakes (2013-2035)

يقدم التطبيق نظامًا تفاعليًا متطورًا لتحليل التغيرات المورفولوجية والزراعية للبحيرات الشمالية في دلتا مصر (المنزلة، البرلس، مريوط، إدكو) بدقة عالية ومرونة فائقة من 2013 إلى 2035. الوظائف الرئيسية: متابعة التغيرات السنوية في مساحة البحيرات، من 2013 إلى 2035. تحليل الإنتاج الزراعي والسمكي: عرض بيانات تفصيلية لإنتاج الأسماك، المزارع، والمحاصيل (القمح، الأرز، القطن، قصب السكر). رسوم بيانية (خطية) لتتبع التغيرات عبر السنوات. بيانات تفصيلية: جدول بيانات ديناميكي يعرض معلومات البحيرات بناءً على اختيار البحيرة.

The application provides an advanced interactive system for analyzing morphological and agricultural changes in Egypt's northern Delta lakes (Manzala, Burullus, Mariout, Edku) with high precision from 2013 to 2035. Key Features: Annual tracking of lake area changes (2013-2035). Agricultural and fish production analysis: Detailed data on fish production, farms, and crops (wheat, rice, cotton, sugarcane). Line charts for tracking changes over years. Detailed data: Dynamic data table displaying lake information based on selection.

عرض ثلاثي الأبعاد 3D Visualization

عرض ثلاثي الأبعاد للتضاريس الأرضية

3D Terrain Visualization

يقدم التطبيق نظامًا تفاعليًا متقدمًا لتحليل بيانات التضاريس ثلاثية الأبعاد باستخدام طبقة ImageryTileLayer بدقة عالية ومرونة فائقة. الوظائف الرئيسية: خريطة ثلاثية الأبعاد تفاعلية: عرض دقيق للتضاريس باستخدام إحداثيات جغرافية وطبقة أساسية (gray-vector) مع إمكانية التبديل إلى طبقة ساتلية. دعم تأثيرات الظل (Shaded Relief) مع خيارات تقليدية ومتعددة الاتجاهات. تخصيص تأثيرات العرض: ضبط زاوية الشمس (Altitude) واتجاهها (Azimuth) لتعديل تأثيرات الإضاءة والظل. تعديل عامل التضخيم (Z-Factor) لتكبير التضاريس وتحسين التصور البصري.

The application provides an advanced interactive system for analyzing 3D terrain data using ImageryTileLayer with high precision and flexibility. Key Features: Interactive 3D Map: Accurate terrain display with geographic coordinates and base layer (gray-vector) with satellite layer option. Shaded Relief effects with traditional and multi-directional options. Display Customization: Sun angle (Altitude) and direction (Azimuth) adjustment for lighting effects. Z-Factor modification for terrain exaggeration and enhanced visualization.

طبوغرافيا Topography

رسم خرائط طبوغرافية دقيقة

Precise Topographic Mapping

التطبيق عبارة عن نظام تفاعلي لتحليل التنقيب ثلاثي الأبعاد حيث توصلت شمال الدلتا باستخدام ArcGIS Maps SDK for JavaScript 4.32 ومكونات الكالسيت . فإنها تهدف إلى دراسة المورفولوجية بطريقة سهلة وسهلة. الوظيفة الرئيسية : خريطة ثلاثية الأبعاد تفاعلية : الفنانة المعروفة باستخدام ImageryTileLayer بدقة عالية. يدعم التبديل بين العودة (Gray-vector) وطبقة ساتلية (هجينة). وأشار إلى إحداثيات الدلتا ([31.0, 31.0]) لتكبير 8. تحليل التضاريس : الارتفاعات (Elevation) : تصور الارتفاعات مقابل متفاوت الفروق في العمق. الانحدارات (المنحدر) : يحسب الانحدار (0-30°) ويعرضها لونها (أخضر للمنخفض، أحمر للعالي).

Main Features: Interactive 3D Map: High-resolution imagery using ImageryTileLayer. Toggle between grayscale vector (Gray-vector) and satellite (Hybrid) layers. Pre-set Delta coordinates view at [31.0, 31.0] zoom level 8. Terrain Analysis: Elevation visualization with depth variations. Slope calculation (0-30°) with color gradient (green for low, red for high slopes).
